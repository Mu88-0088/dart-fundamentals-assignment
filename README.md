## Assignment Overview
This repository contains my solutions for the Dart Fundamentals programming assignment. The project demonstrates proficiency in core Dart concepts including lists, functions, OOP, exception handling, and asynchronous programming.






## Task 1: Number Analysis App
A console application that analyzes a list of integers using custom functions:
- **findMax()** - Finds largest number using manual loop
- **findMin()** - Finds smallest number using manual loop  
- **calculateSum()** - Computes sum using accumulator pattern
- **calculateAverage()** - Computes average by reusing calculateSum()
- **Bonus Features**: Empty list handling, negative counter, bubble sort implementation, built-in method comparison

## Task 2: Async Calculator App
An object-oriented calculator with asynchronous capabilities:
- **Calculator class** with add, subtract, multiply, divide methods
- **Error handling** with ArgumentError for division by zero
- **Async/await** with simulated network delay (1.5 seconds)
- **Exception handling** with try-catch blocks
- **Bonus Features**: Calculation history, chained operations, custom exceptions, parallel execution demo
